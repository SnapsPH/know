# Knowledge Retrieval CLI - Executable Distribution

## Overview
Knowledge Retrieval is a powerful CLI tool for web scraping, data processing, and AI-powered information management.

## System Requirements
- Windows 10 or later (64-bit)
- Minimum 2GB RAM
- Recommended: Local Ollama installation for AI processing

## Installation
1. Extract the ZIP file to any location
2. No additional installation required

## Usage

### Interactive Mode
```bash
knowledge_retrieval.exe
```

### Command-line Options
```bash
# Scrape a URL
knowledge_retrieval.exe --url https://example.com

# Process data
knowledge_retrieval.exe --input-file data.json
```

## Configuration
Edit `config/settings.ini` to customize:
- Scraping settings
- AI processing options
- Ollama model configurations

### Key Configuration Options
- `crawl_depth`: Control web scraping depth
- `ollama_url`: Set Ollama server URL
- `default_model`: Choose default AI model

## Troubleshooting
- Ensure Ollama is running locally
- Check internet connectivity
- Verify configuration settings
